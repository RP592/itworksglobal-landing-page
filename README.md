# IT Works Global - Landing page
An elegant landing page for IT Works Global Update

## Build and run

1. Install the packages:

```sh
npm install
```

2. Run the app:

```sh
npm run start
```

## Switching node versions

- Check node version:

```sh
node -v
```

- Install a specific version:

```sh
nvm install 14
```

- Use the newly installed version

```sh
nvm use 14
```
